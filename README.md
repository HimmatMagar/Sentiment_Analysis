# Sentiment_Analysis

This project provides an end-to-end deep learning implementation for sentiment analysis. It leverages modern machine learning operations (MLOps) practices, including data versioning with DVC, experiment tracking, and a modular pipeline structure for preprocessing, training, and deploying sentiment analysis models.

### Installation Guide
#### Create a conda environment
```bash
conda crete -p env python==3.12 -y
```

#### Activate a conda environment
```bash
conda activate env/
```

#### Install requirements
```bash
pip install -r requirements.txt
```
